# Conjugated-Block-Polyelectrolytes-CBEs-DFT-Structures

DFT outputs supporting the manuscript:

**"Alkali metal borate conjugated block polyelectrolytes as tuneable mixed ionic-electronic conductors"**
Sebastian J. Leeming, Samuel Fryer, Georgina L. Gregory,* _Published in Polymer Chemistry_, **2026**, 17, 1405-1418. [DOI:10.1039/D6PY00119J](https://doi.org/10.1039/D6PY00119J)
Open access (CC-BY).

## **Method**

All calculations were performed with Gaussian 16 (Rev.C.01). Geometry optimisations at the ωB97X-D/6-311G(d,p) level of theory at 298 K, using the CPCM implicit solvent model for DMSO. Additional calculations performed at the ωB97X-D/6-311+G(d,p) level for selected structures to better describe the anionic borate. Frequency calculations at the same level of theory confirmed optimised structures as minima.

Model structures comprise 3-hexylthiophene units (one or three, as indicated), one polyether repeat unit with a pendant oligoether side chain (4 EO units), and one borate-functionalised polycarbonate repeat unit.

## **Contents**

Gaussian 16 `.out` files for optimised structures, P3HT-_block_ P(EO-_graft_ PEG)_block_Polycarbonate, organised by:
- **Ion Coordination Environments:** Li⁺, Na⁺, K⁺ binding to anionic CPE in M-borate, M-borate-ether, M-ether, and M-ether-thiophene environments.
- **Polarons and Bipolarons:** anionic CPE (neutral, polaron, bipolaron, linear and coiled conformations) and Li/Na/K-CPE (neutral, polaron, bipolaron).
